# Static Support Tables

`SUPPLEMENT_COMMANDS.md` lists the command-backed regeneration entry points for the
review bundle. A small number of supporting tables are intentionally included as
static summaries rather than separate command-line entry points.

- Main Table 3 (`tab:ranking-reversal`) is embedded in `sections_ed/05_results.tex`.
  Its values are copied from the same bundled artifacts used for Table 1 and
  the appendix objective-family finetune support table. That support table is regenerated by
  `python -m src.scripts.make_objfam_finetune_support_table`.
- Appendix Tables 10 and 11 are guardrail sensitivity summaries. Their displayed
  support CSVs are bundled at
  `paper/neurips2026_selection_risk/tables/guardrail_uncertainty.csv` and
  `paper/neurips2026_selection_risk/tables/guardrail_budget_sweep.csv`.
  They are supporting sensitivity checks, not separate command-backed entry points
  in the minimal review bundle.
- Appendix Table 12 is an adaptive baseline-only guardrail sensitivity summary.
  It is included as a static support table; the full source-row regeneration path
  is not part of the minimal review bundle.
- Appendix Table 13 is a purposive public-practice audit. It is a manual
  qualitative audit summary, not a prevalence estimate and not a command-regenerated
  quantitative artifact.

These static summaries do not change the advertised regeneration scope: the
commands in `SUPPLEMENT_COMMANDS.md` regenerate the listed command-backed
claim-bearing outputs from saved artifacts without retraining.
